Character Penalty Application Guide

Purpose: Penalties account for increased annoyance beyond simple loudness. Same dB(A) level can be much more annoying with tonal or low-frequency content.

Three Main Penalty Types

Penalty TypeCriteriaAmountHVAC Examples
+5 dB TonalPure tone prominence per ANSI S12.9 Annex C+3 to +6 dB (use +5 typical)63 Hz cooling tower hum, fan blade pass (120 Hz), compressor whine
+5 dB Low-FrequencydB(C) - dB(A) > 20 dB+5 dB31.5-63 Hz rumble penetrates walls, felt as much as heard
+5 dB ImpulsiveRepetitive impacts, clanks, bangs+5 dBCompressor cycling, damper slamming, fan startup transients

How to Determine Penalties from Measurements

StepActionTool/Data Required
1Measure base dB(A) levelSound level meter (ISO/IEC 61672)
2Measure dB(C) or dB(Z) levelSame meter, different weighting
3Calculate dB(C) - dB(A)If > 20 dB â†’ +5 dB LF penalty
4Examine 1/3 octave bandsFFT analyzer or spectrum display
5Check for pure tone (ANSI Annex C)If tone stands out > 5 dB â†’ +5 dB tonal penalty
6Listen for impulsive characterIf repetitive impacts â†’ +5 dB impulsive penalty
7Sum applicable penaltiesTotal = base dB(A) + penalties

Example: San Mateo Science Center

Measured at 3rd-floor residence (proper location):

Base dB(A): 58 dB(A) | dB(C): 81 dB(C) | dB(C) - dB(A) = 23 dB

1/3 octave spectrum shows prominent 63 Hz tone standing 8 dB above adjacent bands

Penalties:

+5 dB (Low-Frequency): dB(C) - dB(A) = 23 dB > 20 dB threshold

+5 dB (Tonal): 63 Hz tone prominence exceeds ANSI criteria

Total: 58 + 5 + 5 = 68 dB(A) vs. 50 dB(A) limit = +18 dB VIOLATION
